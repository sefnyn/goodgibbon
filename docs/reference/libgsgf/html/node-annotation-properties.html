See http://www.red-bean.com/sgf/properties.html#C.

GB (good for white)

See http://www.red-bean.com/sgf/properties.html#GW.

HO (hotspot)

Type: GSGFDouble

Marks a position as "hotspot".

See http://www.red-bean.com/sgf/properties.html#HO.

N (name)

Type: GSGFSimpleText

Gives a name to a position.

See http://www.red-bean.com/sgf/properties.html#N.

UC (unclear)

Type: GSGFDouble

Marks a position as unclear.

See http://www.red-bean.com/sgf/properties.html#UC.

V (value)

Type: GSGFReal

Value of the position. Positive values are given to positions with an advantage for black, negative values given to positions with an advantage for white.

See http://www.red-bean.com/sgf/properties.html#V.
