LIBGSGF Reference Manual

Markup Properties

AR (arrow)

Type: GSGFListOf, item type: GSGFComposed (GSGFPoint:GSGFPoint)

A list of point pairs. Each pair specifies an arrow to draw.

See http://www.red-bean.com/sgf/properties.html#AR.

CR (circle)

Type: GSGFListOf, item type: GSGFPoint.

Mark points with a circle. The properties CR, MA, SL, SQ, and TR must not be used to mark the same point within one node.

See http://www.red-bean.com/sgf/properties.html#CR.

DD (dim point)

Type: Possibly empty GSGFListOf, item type: GSGFPoint.

Dim out points.

See http://www.red-bean.com/sgf/properties.html#DD.

LB (text on point)

Type: GSGFListOf, item type: GSGFComposed (GSGFPoint:GSGFSimpleText)

A list of pairs of points combined with a text label. Points must be unique.

See http://www.red-bean.com/sgf/properties.html#LB.

LN (line)

Type: GSGFListOf, item type: GSGFComposed (GSGFPoint:GSGFPoint)

A list of point pairs. Each pair specifies a line to draw between points.

See http://www.red-bean.com/sgf/properties.html#LN.

MA (mark point with an X)

Type: GSGFListOf, item type: GSGFPoint.

Mark points with an X. The properties CR, MA, SL, SQ, and TR must not be used to mark the same point within one node.

See http://www.red-bean.com/sgf/properties.html#MA.

SL (selected)

Type: GSGFListOf, item type: GSGFPoint.

Mark points as selected. The properties CR, MA, SL, SQ, and TR must not be used to mark the same point within one node.

See http://www.red-bean.com/sgf/properties.html#SL.

MA (mark point with a square)

Type: GSGFListOf, item type: GSGFPoint.

Mark points with a square. The properties CR, MA, SL, SQ, and TR must not be used to mark the same point within one node.

See http://www.red-bean.com/sgf/properties.html#SQ.

MA (mark point with a triangle)

Type: GSGFListOf, item type: GSGFPoint.

Mark points with a triangle. The properties CR, MA, SL, SQ, and TR must not be used to mark the same point within one node.

See http://www.red-bean.com/sgf/properties.html#TR.
