Description

A GSGFFlavor defines how to handle properties for a particular kind of SGF. Different games have different sets of properties. In SGF files, the flavor is defined by the GM property.

It is the base class for other flavors, and defines properties that are common to all specialized SGF flavors.

This GSGFFlavor class is actually internal to libgsgf. You only have to bother about these details if you want to implement your own flavors of SGF.

When you want to add your own flavor, please check the following list mostly taken from http://www.red-bean.com/sgf/properties.html#FF for a reference:

  • 0: No type ("")

  • 1: Go ("GO")

  • 2: Othello ("OTHELLO")

  • 3: Chess ("CHESS")

  • 4: Gomoku+Renju ("GOMOKO+RENJU")

  • 5: Nine Men's Morris ("NINE_MENS_MORRIS")

  • 6: Backgammon ("BACKGAMMON")

  • 7: Chinese Chess ("CHINESE_CHESS")

  • 8: Shogi ("SHOGI")

  • 9: Lines of Action ("LINES_OF_ACTION")

  • etc. See the above link for more flavors

The strings in parentheses define the libgsgf identifiers for these games. The default flavor (in absence of an FF property) is 1 (for Go).

The flavor 0 is not defined by the SGF specification. It is used by libgsgf for properties that are common to all other flavors.

The only flavors currently implemented are 0 and 6 (for Backgammon).
