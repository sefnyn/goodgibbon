gsgf_compose_new ()

GSGFCompose *       gsgf_compose_new                    (GSGFCookedValue *value,
                                                         ...);

Creates a new GSGFCompose from a list of GSGFCookedValue objects. The stored items are "hijacked" and are now considered property of the composed object, and you must not g_object_unref() them yourself.

Note that you must pass cooked values! Passing a GSGFRaw would not make sense since a GSGFRaw can actually be a list of values.

value :

 The first value to store.

... :

 Other optional values, terminated by NULL.

Returns :

 The new GSGFCompose.
