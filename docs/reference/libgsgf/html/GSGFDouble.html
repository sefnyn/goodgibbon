struct GSGFDoubleClass

};

A GSGFDouble can take one of two values, either one or two. The name is a little misleading as most people will rather associate a double-precision floating point value with it. It is, however, a binary object that is either one or two. The value one means "normal" and a value of two means "more than normal". For example, the property GB (good for black) is a GSGFDouble. A value of one means, "good for black", and a value of two means "very good for black".
