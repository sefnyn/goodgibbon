Miscellaneous Properties

FG (figure)

Type: GSGFEmpty or GSGFCompose (of GSGFNumber:GSGFSimpleText)

Time left for white in seconds after the move was made.

See http://www.red-bean.com/sgf/properties.html#FG.

PM (print move numbers)

Type: GSGFNumber

Specifies how move numbers should be printed.

See http://www.red-bean.com/sgf/properties.html#PM.

VW (view part)

Type: Possibly empty GSGFListOf, item type: GSGFPoint.

View only part of the board.

See http://www.red-bean.com/sgf/properties.html#VW.
