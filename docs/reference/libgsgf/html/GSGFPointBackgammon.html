Description

Representation of one single point in Backgammon.

The class is actually internal. You will never have to use it yourself.

gsgf_point_backgammon_new ()

GSGFPointBackgammon * gsgf_point_backgammon_new         (gint point);

Creates a new GSGFPointBackgammon.

Points are counted with base 0, or 0 is the ace point for white (and the 24 point for black), through to 23 which is the 24 point for white (and the ace point for black). 24 is the bar, and 25 is the bearoff tray.

 The point as a number between 0 and 25.
