Description

A GSGFColor encapsulates an SGF color, that is either black or white.

gsgf_color_new_from_raw ()

GSGFCookedValue *   gsgf_color_new_from_raw             (const GSGFRaw *raw,
                                                         const GSGFFlavor *flavor,
                                                         const struct _GSGFProperty *property,
                                                         GError **error);

Creates a new GSGFColor from a GSGFRaw. This constructor is only interesting for people that write their own GSGFFlavor.

The raw parameter should contain the color as a string. Both abbreviated ("B" or "W") and spelled out ("black" or "white") colors are accepted. Case does not matter.

raw :

 A GSGFRaw containing exactly one value that should be stored.

flavor :

 The GSGFFlavor of the current GSGFGameTree.

property :

 The GSGFProperty raw came from.

error :

 a GError location to store the error occuring, or NULL to ignore.

Returns :

 The new GSGFColor or NULL in case of an error.
