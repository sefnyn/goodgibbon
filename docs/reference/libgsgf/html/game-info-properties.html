CP (copyright)

Type: GSGFSimpleText

Copyright information.

See http://www.red-bean.com/sgf/properties.html#CP.

Important

GNU Backgammon uses this property instead of CO for encoding the cube owner. The possible values are w and b for white and black respectively.
