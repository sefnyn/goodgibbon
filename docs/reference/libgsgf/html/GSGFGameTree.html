Implemented Interfaces

GSGFGameTree implements GSGFComponent.

Description

A GSGFGameTree represents a single game in a GSGFCollection.

gsgf_game_tree_get_children ()

GList *             gsgf_game_tree_get_children         (const GSGFGameTree *self);

Get the list of GSGFGameTree objects stored in a GSGFGameTree.

A GSGFGameTree can have other GSGFGameTree instances as children. This is seldom useful.

This list is not a copy. You should not free it. The list becomes invalid, when you add or remove nodes.

 the GSGFGameTree.
