gsgf_move_backgammon_new_from_raw ()

GSGFMoveBackgammon * gsgf_move_backgammon_new_from_raw  (const GSGFRaw *raw,
                                                         GError **error);

Creates a new GSGFMoveBackgammon from a GSGFRaw. raw must be single-valued and contain a string matching the SGF backgammon move syntax (see online docs for more details). Additionally, libgsgf allows the following move specifications: "resign:N" where N is an integer greater than 0 specifying the value of the offered resignation (1 for normal, 2 for gammon, 3 for backgammon), "reject" for rejection of an offered resignation, "accept" for an accepted resignation.

gsgf_move_backgammon_new_from_string ()

GSGFMoveBackgammon * gsgf_move_backgammon_new_from_string
                                                        (const gchar *str,
                                                         GError **error);

Creates a new GSGFMoveBackgammon from a string. The contents of string must match the SGF backgammon string syntax (see online docs for more details). Additionally, libgsgf allows the following move specifications: "resign:N" where N is an integer greater than 0 specifying the value of the offered resignation, "reject" for rejection of an offered resignation, "accept" for an accepted resignation.

Applications should accept any positive value for a resignation. The value of the cube is already taken into account. If a player resigns with a gammon and the cube show 2, the "value" or the resignation should be 4.

str :

 The string to parse.

error :

 a GError location to store the error occuring, or NULL to ignore.

Returns :

 The new GSGFMoveBackgammon.
