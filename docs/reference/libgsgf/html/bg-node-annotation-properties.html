LIBGSGF Reference Manual

Node Annotation Properties

A (Analysis)

Type: GSGFListOf, item type: GSGFSimpleText.

The analysis for a move. The property is not documented.

DA (Doubling Analysis)

Type: GSGFSimpleText.

The analysis of a doubling decision. The property is not documented.

BC (Bad Cube)

Type: GSGFDouble.

Mark a doubling decision as bad. The property is not documented.

The value of the property gives the degree of badness, the higher the worse.

See also DC.

DC (Doubtful Cube)

Type: GSGFEmpty.

Mark a doubling decision as doubtful. The property is not documented.

See also BC.

LU (Luck)

Type: GSGFReal.

Luck measurement of a roll.

MR (Marked As Rolled Out)

Type: GSGFSimpleText.

Undocumented.

Only useful in conjunction with an A property. It contains a space-separated list of the numbers of evaluated possible moves that have been rolled out.

Example: If there are 20 possible moves in a certain position, and the first, the second, and the fourth move have been rolled out in the analysis, the MR property would contain the string "0 1 3". Note that counting starts at zero!
