CO (Cube Position)

Type: GSGFSimpleText

See http://www.red-bean.com/sgf/backgammon.html#CO.

Important

GNU Backgammon uses the CP property instead. This is most probably a bug or typo but it unlikely to change in the future. For compatibility with GNUBG, Gibbon does the same but it additionally sets the correct property CO!
