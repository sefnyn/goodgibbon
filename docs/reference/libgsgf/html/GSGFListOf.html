Description

A GSGFListOf is a list of GSGFCookedValue objects.

The SGF specification has another type elist. This is a list that can possibly be empty. In libgsgf, there is no corresponding type for this. Whether a list can be empty or not is considered to be part of the semantics.

You can detect empty lists by checking the item type with gsgf_list_of_get_item_type(). If it is GSGF_TYPE_EMPTY the list is empty.
