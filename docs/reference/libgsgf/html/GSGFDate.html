Description

A GSGFDate represents a list of dates with at least one value.

The maximum accuracy is one day. If you need more you have to store this externally, for example in the file name.

The class uses GSGFDateDMY for its internal date representation and uses the values G_DATE_BAD_MONTH or G_DATE_BAD_DAY for month or day of the month values not available.
