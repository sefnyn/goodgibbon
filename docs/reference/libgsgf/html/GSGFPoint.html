GSGFPoint — Abstract base class for an SGF point.

Description

The http://www.red-bean.com/sgf/sgf4.html specification does not define the details for a point. The specific point incarnations should inherit from this class.

gsgf_point_get_normalized_value ()

gint                gsgf_point_get_normalized_value     (const GSGFPoint *self);

Get a unique gint identifying the point. This method must be implemented by every GSGFFlavor. It is used to compare points for equality. This is needed in order to make sure that lists of points do not contain duplicate points.
